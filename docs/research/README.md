# Research

Investigations that inform decisions but aren't decisions themselves.

## What lives here

- **Prior-art analyses** — competitive landscape, standards we lean on (e.g., [`prior-art.md`](prior-art.md)).
- **Spike reports** in [`spikes/`](spikes/) — gating investigations whose results unblock or
  redirect a decision (e.g., the isomorphic-git mobile benchmark per ADR-0018 #3).
- **Watch-list notes** — adjacent projects / specs being monitored (Obsidian Web Clipper
  Interpreter, GBrain features, MCP spec evolution).

## What doesn't live here

- Decisions — those are in [`../adr/`](../adr/).
- Implementation contracts — those are in [`../specs/`](../specs/).
- Architecture / philosophy — those are in [`../architecture.md`](../architecture.md) and
  [`../philosophy.md`](../philosophy.md).

## Promotion path

Research → ADR: when an investigation resolves an Open question (ADR-0018), the research
file stays as the supporting evidence; the ADR records the decision and references it.
